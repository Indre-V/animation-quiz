How to Play

  1. Select a Category
  2. Click to Start the Quiz
  3. Four options will be displayed
  4. Select the answer
  5. Go to the next question
  6. Once the last question is answered and click Finish
  7. Score message will be displayed
  8. Select Play Again to go back to the main page
